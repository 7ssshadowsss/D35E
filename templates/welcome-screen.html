News

Class progression

Release 0.86 bring proper class and level progression - no more guessing which skills were class skills on level 5 and how many Hit Points you got. Each gained level can have a class assigned to it, so everything is calculated properly. (This option is turned off for existing Actors)

Learn more about Level progression

Updated Theme

Release 0.86 comes with a first version of redesigned theme - now enabled for all new players!

More theme changes and minor redesigns will come in minor updates in 0.86 line.

Onboarding

We introduced a new onboarding process for players. You can check it yourself - just interact with dialogue box below.

Read more about Onboarding

Magic Items

Magic Items system allows for creating any magic item you want - from simple Longsword +1 to complex ones like Flame Tongue.

Learn more about Magic Items
