DND 3.5E SRD

Welcome to DND 3.5E SRD implementation for Foundry VTT! It was created from love for old DND editions and aims to provide full 3.5E SRD implementation and system flexible enough for anything else.

If you have any questions, you can talk to with me and other users on project Discord

You can check current status of SRD implementation here

If you find any bugs you can post them directly to GitHub issues. You can also post them on dedicated discord channel, but keeping a copy on GitHub will make it easier for anyone to see if issue has already been posted.

Happy GMing,
Rughalt
